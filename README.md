# ShopSaree001
An E-commerce Website selling Sarees.
This website is built using HTML, CSS, BootStrap, and JavaScript.
# Features
1] Has -
1. Home Page 
2. Login Page
3. Products Page
4. Product Page 
5. About Us Page
   
2]Has two modes - Dark and Light

3]Scroll Up button
